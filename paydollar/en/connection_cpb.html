  • Client Post Through Browser

It is the most popular connection method among merchants. The advantage of this connection method is simple and speedy. On the other hand, payment transaction flow is ready to use. Merchant can kick off the web site on-the-fly with just a small scale integration.

Scope and Compatibility

This connectin is designed for merchants who have Online Shopping Cart System. The routine is HTML-based with Javascript and should be widely applicable to on-line shopping cart software and architecture whose technical specifications and varieties are beyond the scope of this document. Compatibility with shopping cart software is yet to be exclusively given and would not be includein the

The compatibility version of the software code is as follow:

Software Code Version
HTML 4.0
Javascript 1.3

The compatibility version of the software code with popular browser software is as follow:

Software Code Version
Microsoft Internet Explorer 11 or above
Mozilla Firefox 27 or above
Google Chrome 30 or above
Safari 7 or above
